Smart Mirror Project

Someday before in the summer of 2020, I watched some Youtube videos and read articles on different raspberry pi uses to try to scratch my itch for creating something. I saw multiple projects on what people called smart mirrors, aka magic mirrors which are mirrors that display time, news, and weather. Furthermore, I stumbled upon a community of smart mirror builders who were very supportive in helping each other out. The mirror and community fascinated me to the point where I needed to build one as well. I started by getting all the items needed for the build.

The part list I planned out consisted of...
  • Wood for frame
  • Wood Paint
  • Acrylic See-Through Mirror
  • Various Screws & Nails
  • Computer Sound Bar
  • Raspberry Pi 4 2GB
  • Raspberry Pi Heatsinks
  • 24" 1080p LED Monitor
  • Micro HDMI -> HDMI Cable
  • Raspberry Pi Power Cable
  • USB Stereo Sound Adapter
  • Amazon Echo Flex
  • USB microSD reader
  • MicroSD Card
  • Power Strip Surge Protector
  • Sonos Smart Switch

After constructing the frame and installing the mirror and monitor, I had to set up the smart mirror software that the raspberry pi will run. Through MagicMirror2, I was able to access a community driven smart mirror platform. With this platform users can contribute to the platform with new ideas and modules for the mirror to display. One module I made sure to add was the spotify music player because music is great. Along with the spotify module, I wanted the mirror to have Alexa control my smart mirror through voice commands. There was a module for Alexa, however I found it to be buggy. So, I simply installed an Amazon Echo inside the mirror which gave the same functionality.
